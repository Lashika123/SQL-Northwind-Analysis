# Northwind SQL Data Analysis Project

## Project Overview
This project contains a comprehensive collection of **SQL queries performed on the Northwind Database** to analyze different business scenarios such as sales performance, customer behavior, employee productivity, product performance, and logistics operations.

The purpose of this project is to demonstrate **SQL data analysis skills used by Data Analysts and Data Scientists** to extract meaningful insights from relational databases.

The project includes **80+ SQL queries** that answer real-world business questions using advanced SQL techniques.

---

## Dataset

The analysis is based on the **Northwind Sample Database**, a popular dataset used for learning SQL and database management.

The database simulates a **trading company that sells products internationally** and contains information about customers, orders, products, employees, and shipping companies.

### Main Tables Used

- Customers  
- Orders  
- Order Details  
- Products  
- Categories  
- Employees  
- Suppliers  
- Shippers  
- Territories  
- Region  

These tables are related through primary and foreign keys to simulate a real-world relational database.

## SQL Concepts Used

This project demonstrates a wide range of SQL concepts used in real-world analytics.

### 1. Data Retrieval
- SELECT statements
- Filtering using WHERE clause
- Sorting results using ORDER BY

### 2. Joins
Used to combine data from multiple tables.

- INNER JOIN
- LEFT JOIN
- Multi-table joins

### 3. Aggregation Functions
Used to summarize data.

- COUNT()
- SUM()
- AVG()

### 4. GROUP BY and HAVING
Used for grouping data and filtering aggregated results.

### 5. Subqueries
Nested queries used to perform advanced filtering.

### 6. Window Functions
Used for ranking and analytical calculations.

- RANK()
- LAG()
- OVER()

### 7. Common Table Expressions (CTE)
Used to simplify complex queries.

### 8. Set Operations
- UNION
